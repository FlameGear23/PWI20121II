- A TAG p serve para colocar parágrafos dentro do código; - A TAG br serve para quebrar linhas; - AS TAGS de h1 até h6 servem para definir o tamanho da fonte dentro do código; - A TAG html serve para indicar o início e o fim do código; - A TAG title serve para mostrar o título; - A TAG styles serve para colocar cor dentro do código; - A TAG background serve para colocar uma cor no fundo no do código; - A TAG color serve pra dar cor nas letras; - A TAG body é usada para definir o corpo do texto; - A TAG head define a parte introdutória da página; - A TAG link é usada para juntar fotos ao código; - A TAG img é usada para introduzir uma foto ao código; - A TAG alt é uma tag que é usada para acessibilidade, quando um deficiente visual passa o cursor por cima da imagem ela é "falada";
